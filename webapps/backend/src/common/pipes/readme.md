# Pipes คือตัวกลางที่ใช้ในการจัดการข้อมูลก่อนที่ข้อมูลนั้นจะส่งไปถึง Handler ของ Route (หรือ Controller) ครับ
หน้าที่หลักของ Pipe มี 2 อย่างที่สำคัญคือ:
    Transformation: แปลงข้อมูลให้อยู่ในรูปแบบที่ต้องการ (เช่น รับ string "10" มาแล้วแปลงเป็น number 10)
    Validation: ตรวจสอบความถูกต้องของข้อมูล (เช่น เช็คว่าอีเมลพิมพ์ถูกรูปแบบไหม หรือค่าที่ส่งมาว่างหรือเปล่า) ถ้าข้อมูลไม่ถูกต้อง Pipe จะโยน Exception กลับไปให้ Client ทันที
